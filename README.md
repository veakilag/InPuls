# InPuls

Текущая сборка: v9. PWA-кеш временно отключён на период активной разработки.

Личный бесплатный браузерный скринер для Binance USDⓈ-M Futures — в ритме рынка. Рыночные данные приходят напрямую из публичных WebSocket-потоков Binance; API-ключи и сервер не нужны.

## Что уже есть

- USDT-фьючерсы с фильтром минимального 24-часового оборота;
- изменение цены за 15 секунд, 1 и 5 минут;
- приблизительный оборот в минуту и ускорение относительно фонового потока;
- сделки в секунду и агрессивные покупки/продажи для самых активных монет;
- поток ликвидаций;
- свечной график с секундными свечами из потока сделок, биржевыми таймфреймами и диапазонами от 15 минут до года;
- правый топ монет по рейтингу, импульсу, обороту, движению и ликвидациям;
- компактный табличный радар до 100 монет и масштабирование графика от позиции курсора через Ctrl + колесо;
- ручное масштабирование шкалы цены справа и времени снизу перетаскиванием;
- свободное перемещение по истории удержанием графика, funding, NATR 1м/5м и корреляция с BTC;
- сигналы «Импульс», «Нож», «Пробой», «Каскад», «Сжатие»;
- изменяемые пороги, избранное, поиск и звуковые уведомления;
- адаптивный PWA-интерфейс, который можно установить как приложение.

## Запуск

```bash
npm start
```

Открыть `http://127.0.0.1:4173`. История за 15 секунд и 1 минуту появляется после соответствующего времени; полный пятиминутный контекст — после пяти минут работы вкладки.

## Тесты

## Публикация

Проект статический: содержимое этой папки можно опубликовать через GitHub Pages без сборки. В настройках Pages указывается публикация из корня основной ветки.

## Ограничения первой версии

- Пока вкладка закрыта, скринер не собирает историю и не присылает сигналы.
- Секундная история ограничена последними агрегированными сделками Binance и накапливается во время открытой вкладки; полная архивная секундная история потребует собственного сервера хранения.
- «Сжатие» — количественный предварительный фильтр по узкому минутному диапазону и ускорению оборота, а не готовая разметка торгового сетапа.
- Ускорение оборота считается по изменению 24-часового rolling-счётчика Binance и может содержать шум.
- Скринер отмечает активность рынка, но не является командой на сделку.
